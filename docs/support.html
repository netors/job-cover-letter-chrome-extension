Quick Start: The extension is super easy to set up. You'll need an API key from either OpenAI or Claude (or both for maximum flexibility).

Step-by-Step Setup

Install the Extension

Add it to Chrome from the Chrome Web Store (coming soon) or load it manually in developer mode.

Get Your API Key

Choose your AI provider:

  • OpenAI: Visit platform.openai.com/api-keys
  • Claude: Visit console.anthropic.com/keys
Configure the Extension

Click the extension icon ‚Üí Settings gear ‚Üí Add your API key ‚Üí Test connection

Add Your Info

Upload your resume and fill in your contact details. This makes the cover letters personalized.

Start Generating!

Go to any job site, find a job you like, click the extension icon, and hit "Generate Cover Letter"

Pro Tip: Set up both OpenAI and Claude APIs. If one is down or you hit rate limits, you can instantly switch to the other!

How much does this cost? +

The extension is completely free! You only pay for the AI API usage, which is typically:

  • OpenAI GPT-4o-mini: ~$0.01-0.03 per cover letter
  • OpenAI GPT-4o: ~$0.05-0.15 per cover letter
  • Claude 3.5 Sonnet: ~$0.03-0.08 per cover letter
  • Claude 3 Haiku: ~$0.01-0.03 per cover letter

Way cheaper than subscription services that charge $10-30/month!

Which job sites does it work on? +

Pretty much everywhere! I've tested it on:

  • LinkedIn Jobs
  • Indeed
  • Glassdoor
  • Monster
  • ZipRecruiter
  • AngelList/Wellfound
  • Company career pages
  • University job boards
  • Government job sites

If it has a job description on the page, the extension can probably read it!

Is my data secure? +

Absolutely! Here's how your data is handled:

  • API Keys: Stored locally in Chrome's encrypted storage
  • Personal Info: Never leaves your browser except for AI API calls
  • Job Data: Processed temporarily, not stored permanently
  • No Tracking: Zero analytics, no data collection
  • Open Source: You can see exactly what the code does

I built this for myself first - privacy was a top priority.

Can I customize the writing style? +

Yes! The extension offers several options:

  • Writing Tone: Professional, Enthusiastic, Concise, or Friendly
  • Length: Short, Medium, or Detailed
  • Custom Instructions: Add your own guidance in settings
  • Multiple Models: Different AI models have different writing styles

You can also regenerate letters with different settings until you find one you love.

What if the cover letter is terrible? +

I won't lie - sometimes the AI has an off day. Here's what to do:

  • Try a different model: Switch from GPT-4o-mini to GPT-4o or Claude
  • Update your resume: More detailed resume = better letters
  • Add custom instructions: Tell the AI what to emphasize
  • Edit the output: The generated text is fully editable
  • Regenerate: Hit the button again for a fresh attempt

In my experience, GPT-4o and Claude 3.5 Sonnet produce the best results.

Can I use this for other types of writing? +

Technically yes, but it's really optimized for cover letters. The extension:

  • Looks specifically for job descriptions
  • Uses prompts designed for cover letter writing
  • Includes resume context automatically
  • Formats output as business letters

If you need other types of AI writing, I'd recommend using ChatGPT or Claude directly.

üîß Troubleshooting

Extension Not Detecting Jobs

Most Common Issue: This usually happens on dynamic job sites that load content after the page loads.

Try this:

  • Refresh the page after it fully loads
  • Wait a few seconds, then click the extension icon
  • Use the "üîÑ Force Re-detect" button in the popup
  • Make sure you're on the actual job description page, not search results

API Errors

Error: Invalid API key Error: Rate limit exceeded Error: Insufficient quota

Solutions:

  • Invalid API key: Double-check you copied the key correctly
  • Rate limits: Wait a few minutes or switch to the other AI provider
  • Insufficient quota: Add billing to your OpenAI/Claude account

Generated Letters Are Generic

This usually means your resume isn't detailed enough. Try:

  • Adding specific achievements with numbers
  • Including relevant skills and technologies
  • Adding custom instructions about what to emphasize
  • Using GPT-4o or Claude 3.5 Sonnet for higher quality

PDF Download Not Working

This is usually a browser popup blocker issue:

  • Allow popups for the job site domain
  • Try right-clicking the PDF button and "Save link as"
  • Copy the text and paste into Google Docs, then export as PDF
Still having issues? Check the browser console (F12 ‚Üí Console) for error messages, then contact support with the details.
