Privacy Policy

AI Cover Letter Generator Chrome Extension

Last Updated: December 2024
TL;DR: I don't collect your data. Everything stays on your device except for the job descriptions and resume content you send to AI providers. You use your own API keys, you pay your own bills, and I'm not responsible for any costs or issues.

1. Information We Collect

Information We DO NOT Collect

I want to be crystal clear about this:

  • No personal information is collected by me or this extension
  • No usage analytics or tracking
  • No API keys are stored on external servers
  • No cover letters are stored externally
  • No browsing history or job search activity
  • No email addresses, names, or contact information
  • No cookies or tracking pixels

Information Stored Locally

The extension stores the following data on your device using Chrome's secure storage:

  • Your OpenAI and/or Anthropic API keys (encrypted by Chrome)
  • Your personal information (name, email, phone) for cover letter headers
  • Your resume content or uploaded resume file
  • Generated cover letters history
  • Extension preferences and settings

2. How Your Data is Used

Local Processing

All data processing happens locally in your browser. The extension:

  • Reads job descriptions from web pages you visit
  • Combines job data with your resume content
  • Sends this combined data to AI providers for cover letter generation
  • Stores generated letters locally for your reference

External API Calls

When you generate a cover letter, the extension sends the following to your chosen AI provider:

  • Job description text from the current web page
  • Your resume content
  • Your personal information (for letter formatting)
  • Generation preferences (tone, length, etc.)
Important: This data is sent directly from your browser to OpenAI or Anthropic using YOUR API key. I have no access to this data or these requests.

3. Third-Party Services

AI Providers

This extension integrates with:

  • OpenAI (ChatGPT): Subject to OpenAI's Privacy Policy
  • Anthropic (Claude): Subject to Anthropic's Privacy Policy

When you use these services:

  • Your job description and resume data is processed by the AI provider
  • You are responsible for compliance with their terms and policies
  • Data retention and usage is governed by their policies, not mine
  • You pay them directly through your API usage

Job Sites

The extension reads content from job sites you visit, but:

  • No data is sent back to job sites
  • Job sites may have their own tracking and privacy policies
  • The extension doesn't modify or interact with job site functionality

4. Data Security

Local Storage Security

  • API keys are stored in Chrome's encrypted sync storage
  • All data remains on your device unless explicitly sent to AI providers
  • No external databases or cloud storage used by this extension
  • Chrome's built-in security protects your stored data

Transmission Security

  • All API calls to OpenAI/Anthropic use HTTPS encryption
  • No man-in-the-middle interception by this extension
  • Direct communication between your browser and AI providers

5. Your Rights and Controls

Data Control

You have complete control over your data:

  • Delete Data: Remove the extension to delete all local data
  • Export Data: Copy your cover letter history before uninstalling
  • API Key Management: Add, remove, or change API keys anytime
  • History Management: Delete individual cover letters or clear all history

Opt-Out

To stop all data processing:

  • Disable or uninstall the Chrome extension
  • Remove API keys from extension settings
  • Clear Chrome extension data in browser settings

6. Children's Privacy

This extension is not intended for children under 13. I do not knowingly collect personal information from children under 13. If you are under 13, please do not use this extension.

7. Open Source Transparency

This extension is open source, which means:

  • You can inspect all code on GitHub
  • You can verify what data is collected and how it's used
  • You can build and modify the extension yourself
  • Community contributions are welcome

8. Changes to This Policy

I may update this privacy policy from time to time. When I do:

  • The "Last Updated" date will be revised
  • Significant changes will be announced on GitHub
  • Continued use constitutes acceptance of changes

9. International Users

This extension can be used worldwide. However:

  • AI providers may have geographic restrictions
  • You are responsible for compliance with local privacy laws
  • Some job sites may not be accessible in all countries
  • API costs may vary by region

10. Contact Information

If you have questions about this privacy policy:

  • GitHub Issues: Report privacy concerns
  • Code Review: Inspect the source code for data handling practices
Remember: This is a free, open-source tool built by one person. I'm not a big company with lawyers and compliance teams. Use common sense when entering personal information, and always review generated content before submitting it to employers.
